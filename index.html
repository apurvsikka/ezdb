Welcome to EzDB

EzDB is a lightweight file-based key-value database for Node.js, designed for simplicity, speed, and reliability.

Key Features

  • Fast and efficient binary storage
  • Easy-to-use API
  • Indexing for rapid lookups
  • Thread-safe with async mutex
  • Minimal dependencies

Installation

npm install ezdb-main

Getting Started

Create and use a database instance:

import EzDB from "ezdb-main";
const db = new EzDB<{ name: string; age: number }>("users");

API Reference

Insert

Adds a new record.

const user = await db.insert({ name: "Alice", age: 25 });
console.log(user);

Find

Search for records.

const users = await db.find({ age: 25 });

Update

Modify a record.

await db.update(user.id, { age: 26 });

Delete

Remove a record.

await db.delete(user.id);

Clear

Wipe all data.

await db.clear();

Storage Format

EzDB saves data in binary format (.bin) with an index file (.idx).
